🎬 Netflix Clone

A Netflix-inspired web application built using **HTML, CSS, and JavaScript**. The project uses the **TMDB API** to dynamically display movies and allows users to view movie details and watch trailers.

 Features

- Responsive Netflix UI
- Hero Banner
- Dynamic Movie Cards
- TMDB API Integration
- Movie Details Page
- Trailer Playback
- Smooth Animations

 Tech Stack

- HTML5
- CSS3
- JavaScript
- TMDB API

📌 Future Improvements

- Search Movies
- My List
- User Authentication
- Genre Filtering
- Angular Version
